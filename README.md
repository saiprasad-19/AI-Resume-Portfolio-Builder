# 🚀 AI Resume & Portfolio Builder

An AI-powered web application that generates ATS-friendly resumes, professional cover letters, and portfolio content using Groq's Llama 3 model.

## 🌐 Live Demo
[Live Application](https://ai-resume-portfolio-builder-cdpdnxsvhnvueq6yhuaclm.streamlit.app/)

## 📂 GitHub Repository
[GitHub Repository](https://github.com/saiprasad-19/AI-Resume-Portfolio-Builder)

---

## 📖 Overview

AI Resume & Portfolio Builder helps students and job seekers quickly create professional career documents. Users can generate resumes, cover letters, and portfolio content based on their information and download ATS-friendly resumes as PDFs.

---

## ✨ Features

- 📄 ATS-Friendly Resume Generation
- 📝 AI-Powered Cover Letter Generation
- 🌐 Portfolio Content Creation
- 📥 PDF Resume Download
- 🎨 Modern Responsive UI
- ⚡ Powered by Groq + Llama 3

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API (Llama 3)
- ReportLab
- Git & GitHub
- Streamlit Cloud


## ⚙️ Run Locally

```bash
git clone https://github.com/saiprasad-19/AI-Resume-Portfolio-Builder.git
cd AI-Resume-Portfolio-Builder

pip install -r requirements.txt

streamlit run app.py
```

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```
