# AI Resume & Portfolio Builder

An AI-powered web application that generates ATS-friendly resumes, cover letters, and portfolio content using Groq LLMs.

## Features
- Resume Generation
- Cover Letter Generation
- Portfolio Content Generation
- ATS-Friendly Resume Formatting
- PDF Download Support
- Responsive Modern UI

## Tech Stack
- Python
- Streamlit
- Groq API
- ReportLab
